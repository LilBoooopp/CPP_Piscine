This repository contains my solutions for the **C++ Piscine** modules at 42. The goal of these modules is to learn Object-Oriented Programming (OOP) using **C++98**, starting from the basics of namespaces and classes to advanced concepts like templates and the STL.

## üìÇ Modules Overview

| Module | Topic | Key Concepts |
| :--- | :--- | :--- |
| **CPP 00** | **Introduction to C++** | Namespaces, Classes, Member Functions, Stdio Streams, Initialization Lists, Static, Const. |
| **CPP 01** | **Memory Allocation** | `new`/`delete`, References, Pointers to Members, Switch Statement, File I/O. |
| **CPP 02** | **Ad-hoc Polymorphism** | Operator Overloading, Orthodox Canonical Class Form, Fixed Point Numbers. |
| **CPP 03** | **Inheritance** | Inheritance basics, Access Specifiers, Diamond Problem, Virtual Inheritance. |
| **CPP 04** | **Subtype Polymorphism** | Virtual Functions, Abstract Classes, Interfaces, Pure Virtual Functions, Deep Copies. |
| **CPP 05** | **Repetition & Exceptions** | Try/Catch Blocks, Standard Exceptions, Custom Exceptions. |
| **CPP 06** | **C++ Casts** | Scalar Conversion, `static_cast`, `dynamic_cast`, `reinterpret_cast`, `const_cast`, Serialization. |
| **CPP 07** | **Templates** | Function Templates, Class Templates. |
| **CPP 08** | **Templated Containers** | STL Containers (`vector`, `list`, `map`), Iterators, Algorithms (`easyfind`). |
| **CPP 09** | **STL** | Advanced STL usage, Bitcoin Exchange, Reverse Polish Notation (RPN), Merge-Insertion Sort (Ford-Johnson). |

## üõ†Ô∏è Requirements & Rules

- **Language Standard:** C++98.
- **Compiler:** `c++` (clang++ or g++).
- **Flags:** `-Wall -Wextra -Werror -std=c++98`.
- **Forbidden:** No external libraries (e.g., C++11 features, Boost) unless explicitly allowed in the subject.
- **Style:** strict adherence to standard **Orthodox Canonical Class Form** (starting from Module 02).
